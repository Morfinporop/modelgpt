# 🧠 MoSeek Neural Network

Полноценная нейросеть для генерации и анализа изображений с собственной моделью на базе TensorFlow.js

## ✨ Возможности

### 🎨 Генерация изображений
- Создание изображений по текстовому описанию
- Автоматическое улучшение на основе ключевых слов
- Цветовые фильтры (небо, закат, лес, ночь и т.д.)
- Процедурная генерация с использованием нейросети

### 🔍 Анализ изображений
- Определение доминантных цветов
- Оценка качества (0-100%)
- Определение настроения (энергичное, спокойное и т.д.)
- Распознавание типов (пейзаж, портрет, животное и т.д.)
- Детальное описание изображения

### 💬 Умный диалог
- Понимание намерений пользователя
- Контекстные ответы
- Поддержка русского и английского языка
- История разговоров

## 🏗️ Архитектура модели MoSeek IMG v1

### Энкодер (Text Understanding)
- Embedding Layer: 1000 dim → 64 dim
- LSTM: 128 units
- Dense: 32 units с ReLU активацией

### Генератор изображений
- Input: 64-мерный latent vector
- Dense layers: 256 → 512 → 1024 → 2048 → 49152
- Output: 128x128x3 RGB изображение
- Активация: tanh для финального слоя

### Анализатор изображений
- CNN с тремя сверточными слоями
- Filters: 32 → 64 → 128
- MaxPooling после каждого слоя
- Dense classifier: 128 → 10 категорий
- Softmax для вероятностей

## 📦 Деплой на Railway.com

### Автоматический деплой:

1. **Подключите репозиторий**
   - GitHub: `Morfinporop/MoSeek`
   - Branch: `main`

2. **Настройки автоматически применятся**
   - Порт: 8080
   - Node.js: 20.x
   - Build command: `npm run build`
   - Start command: `npm start`

3. **Домен**
   - `mogpt.up.railway.app`

### Файлы конфигурации:

- `railway.json` - настройки Railway
- `Procfile` - команды запуска
- `.nvmrc` - версия Node.js
- `server.js` - Express сервер
